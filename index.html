我是谁？

一个在杭工作的胡建人，比王力宏鼻子大，比坎德拉玛白，比林更新脸长，正经介绍请参见我的简历。大部分时间里，我作为一个数据工程师投身于阿里巴巴的组织数字化事业，每天研究如何搭建高效的数据生产链路，会在 GitHub 和Kaggle上出没。我学软件工程出身，但兴趣和职业都是用数据讲故事，尤其喜欢用可视化方法发现和常识不同的事实。我的业余爱好常常伴随着海浪和雪花，享受在单板上与风对话的畅快感。在泛娱乐年代，更喜欢享受纸质书，好电影和第九艺术带来的美感。

终身学习者与生活体验家

建立这个博客的意义在于希望给世界留下些东西，复盘自己经手技术项目，记录生活中思考时刻，塑造自己的成长型思维。我是一个喜欢折腾的人，爱看科幻小说（心头好是阿西莫夫基地和特德·姜你一生的故事）；喜欢捣弄技术类的东西，比如把玩云服务器，亦或是个体仿真模拟复现的OpenAI的沙盘游戏；也热衷于电子游戏鉴赏，最爱奇异人生系列和我的世界。从初中就开始滑板的我，喜欢HIPHOP文化愿意结交大佬一起解锁新姿势；也爱画画可以说擅长手绘+编程的视觉传达；另外本人沉迷电影无法自拔，最爱星际迷航系列和爱乐之城，看图识电影比GGoogle还准。

优雅永不过时

我属于在某些小处的略微偏执狂，比如网页的字体要调成思源宋体，喜欢井井有条的一切，还在不断修炼自己的全局观。享受在世界不同地点做有趣的事，比如在欧洲用法棍打过架，在新西兰和炸鱼薯条对过话，在英国穿过九磅十五便士的衬衫，在洛杉矶用莫吉托治疗中暑，年纪越大越喜欢荒诞感（今年一定要看完百年孤独）
